# Color js
Parse &amp; convert colors in JS

```JS

var c = new Color("hsva(123, 82%, 100%, 0.5)");
        new Color("hsv(123, 82%, 100%)");
        new Color("hsla(123, 100%, 59%, 0.5)");
        new Color("hsl(123, 100%, 59%)");
        new Color("rgba(46, 255, 56, 0.5)");
        new Color("rgb(46, 255, 56)");
        new Color("#2eff38");
        
c.getHSV();
c.getHSVa();
c.getHSL();
c.getHSLa();
c.getRGB();
c.getRGBa();
c.getHex();

```
